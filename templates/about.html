About This Tool

The Wikidata Reference Validator helps you validate external references of Wikidata items to check if they are alive or dead.

✨ Features:

  • Search Wikidata items using autocomplete.
  • Check references and see if they are alive ✅ or dead ❌.
  • Quickly navigate to Wikidata to edit broken references.

Built with ❤️ using Wikidata API, Bootstrap, and a custom backend.
