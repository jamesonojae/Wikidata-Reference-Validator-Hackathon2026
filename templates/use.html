How to Use the Tool

The Wikidata Reference Validator helps you check whether external references used in Wikidata statements are still accessible or have become dead links. This makes it easier to review and maintain reliable references on Wikidata.

🛠️ Steps to use the tool:

  1. Start typing the name or ID of a Wikidata item in the search box on the homepage. The tool provides autocomplete suggestions directly from Wikidata.
  2. Select the correct item from the suggestions and click Validate References.
  3. The tool will check the external references attached to statements of that item.
  4. Results will show the reference links and indicate whether they are:
    • ✅ Alive – the reference link is still accessible.
    • ❌ Dead – the reference link is no longer accessible.
  5. If a reference is marked as dead, click on the Details button to expand more information about that reference.
  6. You can then review the statement on Wikidata and update the dead reference with a corrected or more reliable source.
  7. When available, the tool may also suggest an archived version of the reference, such as a Wayback Machine snapshot, which can be used to replace or supplement the dead link.
