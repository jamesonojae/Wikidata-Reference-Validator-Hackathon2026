How to Use the Tool

The Wikidata Reference Validator helps editors quickly check whether external references used in Wikidata statements are still accessible or have become dead links. This helps maintain the reliability and verifiability of Wikidata.

Steps to use the tool:

  1. Start typing the name or ID of a Wikidata item in the search box on the homepage. The tool provides autocomplete suggestions directly from Wikidata.
  2. Select the correct item from the suggestions and click Validate References.
  3. The tool will retrieve all statements with references for that item and check whether the reference links are accessible.
  4. Results will display the reference links with their status:
    • ✅ Alive — the link is still accessible.
    • ❌ Dead — the link is no longer reachable.
  5. When a link is marked as dead, you can click on the Details button to inspect the reference and navigate to Wikidata to update or replace it.
  6. When available, the tool may suggest an archived version of the reference (for example from Internet Archive) which can be used to replace the dead link.

💡 Tip: When updating references on Wikidata, consider adding an archive URL and archive date so the reference remains verifiable even if the original page disappears.
